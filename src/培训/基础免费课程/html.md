### 文档声明

```html
<!DOCTYPE html>
```

不写文档声明，将导致浏览器进入怪异渲染模式

### html 根元素

```html
<html lang="en,zh">
    根标签、一个html页面只能有一个跟标签
</html>
```

lang 告诉搜索引擎爬虫该网页用的是什么语言写的，我们的网站是干什么的,SEO的一部分

html5中，不写html元素也是符合标准的

现在一般用

```html
<html lang=cmn-hans></html>
```

### head 文档头

```html
<html>
    <head>
        给浏览器看的
    </head>
</html>
```

### meta文档元数据

可以理解成附加信息

```html
<html>
    <head>
        <meta charset="utf-8">
        <meta content="服装" name="keywords">
        <meta content="这是一个你穿了就不想脱的衣服" name="description">
    </head>
</html>
```

charset：指定网页字符编码

编码字符集：GB2312（只认识简体字符集）、gbk（认识繁体）、unicode（万国码）、utf-8（升级版的万国码）

SEO搜索引擎

```html
<meta http-equiv="X-UA-Compltible" content="ie-edge">
```

告诉浏览器，如果是IE浏览器的话，就使用edge内核。现在IE浏览器支持切换内核

### title

```html
<html>
    <head>
        <meta charset="utf-8">
        <title>淘宝网，淘！</title>
    </head> 
</html>
```

### body文档体

```html
<html>
    <body>
        给人看的
    </body>
</html>
```

默认情况下，body的margin为8px

```html
<html>
    <body>
        <div>
            华府丹郡卡换手机号分开很大回复几乎都是
        </div>
        <div>
            hhfjdhsakjhjfhjdkhsjkahjfhdhhfjkdh
        </div>
    </body>
</html>
```

空格：英文单词分隔符，连续的字母会被当做一个单词，不会换行

### 元素性质4.01及以下版本中

##### 行级元素

1. 内容决定元素所占位置
2. 不可以通过css改变宽高

##### 块级元素

1. 独占一行
2. 可以通过css改变宽高

##### 行级块元素

1. 内容决定大小
2. 可以通过css改变宽高

其他的就是一些html中的标签，不需要记笔记

### HTML5

> 到了html5，已经不叫行级元素和块级元素了，呗内容类别代替了

### 元素的包含关系

1. 容器元素中可以包含任何元素
2. a 元素中几乎可以包含任何元素
3. 某些元素有固定的子元素
4. 标题元素和段落元素不能相互嵌套，并且不能包含容器元素

### 站内资源和站外资源

站内资源：当前网站的资源

站外资源：非当前网站的资源

### 绝对路径和相对路径

站外资源：绝对路径

站内资源：相对路径、绝对路径

绝对路径的书写格式

```
协议名://主机名:端口/path
schema://host:port/path
```

当绝对路径的协议与当前页面的协议是相同的时候，协议可以省略，协议名和冒号（http:）可以省略

协议名：http、https、file(html从本地双击打开会是file协议)

主机名：域名、IP地址

端口号：如果协议是http协议，默认端口号为80；如果协议是https，默认端口号是443。默认端口是可以省略的

相对路径书写格式

以 ./  开头，表示当前资源的文件目录

可以用 ../ 返回上一级目录

相对路径中 ./ 是可以省略的
